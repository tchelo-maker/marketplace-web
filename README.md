# 🛒 Movix - Plataforma de Compra e Venda

Movix é uma aplicação web de e-commerce desenvolvida com HTML, CSS e JavaScript.  
O projeto simula uma plataforma de compra e venda com listagem de produtos e carrinho de compras.

---

## 🚀 Funcionalidades

- 📦 Listagem de produtos
- 🛒 Carrinho de compras
- ⚙️ Interações com JavaScript (DOM)
- 💾 Armazenamento local com LocalStorage (se tiver)
- 📱 Layout responsivo (se tiver)

---

## 🧠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco em prática de lógica de programação, manipulação do DOM e estruturação de aplicações front-end.
